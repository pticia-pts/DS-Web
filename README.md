**MVT - Model,Views e Template**

Model = banco de dados

Views = lógica do sistema

Template = Visual (HTML)

##
**HTTP Request (requisição)** = É o que o cliente envia para o servidor.

*Principais métodos:

GET → buscar dados
POST → enviar dados
PUT → atualizar dados
DELETE → deletar dados

**HTTP Response (Resposta)** = É o que o servidor retorna para o cliente.

*Códigos de status:

200 → OK (deu certo)

404 → Não encontrado

500 → Erro no servidor

403 → Acesso negado

##
## 💻 Alguns comandos:

1- Ativa o ambiente virtual : venv\Scripts\activate;

2- pip install django (instalar django);

3- Cria projeto: django-admin startproject projeto . 

4- Inicia o servidor: python manage.py runserver;

5- Cria um superusuário: python manage.py createsuperuser.
